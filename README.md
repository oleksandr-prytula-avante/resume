# Resume Website

Minimal resume website built with React + Tailwind (Vite).

## Develop

```bash
npm install
npm run dev
```

Edit your content in `src/resume/data.ts`.

## Build

```bash
npm run build
npm run preview
```

## Deploy (GitHub Pages)

This repo includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.

1. Push to the `main` branch.
2. In GitHub: Settings → Pages → set "Build and deployment" to "GitHub Actions".
3. The site will be available at `https://oleksandr-prytula-avante.github.io/resume/`.
